## 要做的事

### 1. 删除 _mock_llm (约第 73-102 行)
整个 `_mock_llm` 方法体，包括 docstring "Return canned responses for testing..."

### 2. 新增 _call_llm 方法
```python
async def _call_llm(self, system_prompt: str, user_message: str) -> str:
    """Call real LLM via DeepTutor's complete() function."""
    try:
        from deeptutor.services.llm import complete
        response = await complete(
            prompt=user_message,
            system_prompt=system_prompt,
        )
        return response
    except Exception as e:
        import logging
        logging.getLogger(__name__).error(f"LLM call failed: {e}")
        return json.dumps({"error": f"LLM call failed: {e}"})
```

### 3. 所有 stage handler 方法签名加 async
因为这 12 个 handler 现在调用 `await self._call_llm(...)`，它们的签名必须从 `def` 改为 `async def`：
- `_run_diagnostic_phase1` → `async def`
- `_run_diagnostic_phase2` → `async def`
- `_run_metacognitive_intro` → `async def`
- `_run_plan` → `async def`

- `_run_module_test` → `async def`

### 4. 替换调用
每个 handler 中的 `self._mock_llm(...)` 改为 `await self._call_llm(...)`。

例如：
```python

# After

```
